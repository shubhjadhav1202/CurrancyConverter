Project Name: Currency Converter (React + Frankfurter API)

Description:
A responsive web application that allows users to convert currencies in real time using the [Frankfurter API]. The app provides a clean UI built with React and Tailwind CSS, enabling users to select source and target currencies, enter an amount, and instantly view the converted value.

Key Features:

Live currency data: Fetches currency codes and exchange rates from the API.

Swap currencies: Quick swap between “from” and “to” currencies.

Favourite conversions: Save and manage frequently used conversions.

Responsive design: Mobile-friendly interface styled with Tailwind CSS.

Error handling: Graceful fallback in case of API errors.

Tech Stack:

Frontend: React.js, Tailwind CSS

API: Frankfurter API for currency data

State Management: React Hooks (useState, useEffect)

MY WORK:-
Designed and implemented the full UI using React.

Integrated the API for fetching currencies and conversion rates.

Added favourites functionality and user interaction features.

Impact:

Reduced manual effort for quick currency conversions.

Provided an intuitive, lightweight, and real-time currency conversion tool.
